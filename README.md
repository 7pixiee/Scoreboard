# Basketball Scoreboard

A simple and interactive basketball scoreboard built using HTML, CSS, and JavaScript.

## Features

* Track scores for both Home and Guest teams
* Add 1, 2, or 3 points with a single click
* Dynamic score updates using JavaScript DOM manipulation
* Clean and responsive user interface
* Custom digital scoreboard font for an authentic sports scoreboard look

## Technologies Used

* HTML5
* CSS3 (Flexbox)
* JavaScript (DOM Manipulation)

## What I Learned

While building this project, I practiced:   

* Using JavaScript functions and parameters
* Updating content dynamically with the DOM
* Creating interactive user interfaces
